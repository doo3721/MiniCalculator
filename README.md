# Mini Calculator


**이 프로젝트는 오토마타와 컴파일러 과제를 위한 것입니다**

**코드는 "만들면서 배우는 인터프린터"에 있는 전자 계산기 프로그램을 참고하였습니다**

---


**현재 진행상황**
 * 프로젝트 생성
 * 연산식에 Power(^) 추가
 * 주석 코멘트 무시(//)
